### 🚀 Projetos em Destaque

Alguns projetos selecionados que destacam meus trabalhos e habilidades recentes:

1. 💈 **[sistema-barbearia](https://github.com/MarcioFerreiraL/sistema-barbearia)**
   - **Descrição:** Um sistema completo e moderno para agendamentos de horários para barbearias. A interface foi construída com Next.js e TypeScript, enquanto a lógica e orquestração backend são garantidas pelo Spring Boot empacotado em containers Docker. Projeto da cadeira de Programação Web, utilizando conceitos de DDD e opiniões de barbeiros para o desenvolvimento da aplicação
   - **Stack:** `Java 21` (Spring Boot 4), `React`, `Next.js`, `TypeScript`, `Tailwind CSS`, `Docker`, `Postgresql`.

2. 🔧 **[sistema-assistencia-tecnica](https://github.com/MarcioFerreiraL/sistema-assistencia-tecnica)**
   - **Descrição:** Um Sistema de Gestão de Assistência Técnica **Full Stack**. Este projeto se destaca por uma abordagem robusta de **Engenharia de Software**, aplicando **Padrões de Projeto GoF** (State Pattern para garantir a segurança da máquina de estados na Ordem de Serviço e Template Method para padronizar os fluxos de reparo técnico).
   - **Stack:** `Java 21` (Spring Boot 3), `PostgreSQL`, `Vue.js 3`, `Vite`.

3. 🏨 **[sistema-hotelaria](https://github.com/MarcioFerreiraL/sistema-hotelaria-java)**
   - **Descrição:** Meu projeto desenvolvido para a disciplina de Programação II, um sistema de gestão hoteleira estruturado sobre os conceitos de Orientação a Objetos (OO).
   - **Stack:** `Java`.
